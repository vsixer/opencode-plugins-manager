# opencode-plugins-manager

> 🇬🇧 [English version](./README.md)

TUI-плагин для [OpenCode](https://github.com/opencode-ai/opencode), который отображает установленные плагины в боковой панели и предоставляет интерактивный интерфейс управления ими.

## Возможности

- **Панель в сайдбаре** — список активных плагинов с версиями
- **Slash-команда `/plugin-manage`** — интерактивный выбор плагина для управления
- **Информация о плагине** — версия, источник (`npm` / `server`) и описание из npm registry
- **Проверка обновлений** — сравнивает установленную версию с последней в npm и предлагает обновить одним нажатием
- **Включение / Выключение** — активировать или деактивировать плагин в рантайме без перезапуска OpenCode
- Читает список плагинов из всех конфигов OpenCode: глобального `~/.config/opencode/opencode.json(c)` и проектного `opencode.json(c)`
- Определяет установленные версии из кэша Bun (`~/.cache/opencode/packages/`)

## Установка

Это **TUI-плагин** — добавлять нужно в `tui.json`, **не** в `opencode.json`.

### Из npm

```json
{
  "$schema": "https://opencode.ai/tui.json",
  "plugin": ["opencode-plugins-manager"]
}
```

### Из локальной папки

```json
{
  "$schema": "https://opencode.ai/tui.json",
  "plugin": ["/абсолютный/путь/до/opencode-plugins-manager"]
}
```

## Использование

### Сайдбар

Секция **Plugins** появляется автоматически в боковой панели OpenCode. Каждая строка показывает имя плагина и его версию. Клик по строке открывает контекстное меню плагина.

### Команда `/plugin-manage`

Введите `/plugin-manage` в поле ввода чата, чтобы открыть список плагинов. Выберите плагин — откроется контекстное меню с действиями:

| Действие | Описание |
|---|---|
| **Инфо** | Версия, источник и описание из npm |
| **Обновить** | Проверить наличие новой версии и установить её |
| **Включить** | Активировать отключённый плагин в рантайме |
| **Выключить** | Деактивировать работающий плагин в рантайме |

## Требования

| Peer-зависимость | Версия |
